# Requirements

- Arch Linux (fresh install supported)
- `sudo` access
- Internet access
- Optional: GPU/network interface parity for exact behavior
- Optional secrets (manually provide):
  - `~/.env` for voice assistant API keys
  - Any project-specific credentials referenced by scripts

Notes:
- This repository mirrors this machine's explicit package set in `manifests/`.
- AUR packages are listed in `manifests/aur-explicit.txt` and installed via `yay`.
